What's Inside

Everything you need
to play Minecraft.

Built with Java Swing, CraftLaunch runs on Windows, macOS, and Linux with no extra software needed beyond a JRE.

🗂
Profile Manager
Create and switch between multiple player profiles — survival, creative, SMP, and more — without re-entering settings.
⚡
Version Switcher
Jump between any Minecraft release from 1.8.9 to 1.21.4, LTS builds, or the latest snapshots in a single click.
🧠
RAM Control
Slide to allocate 1–16 GB of RAM to the JVM. No more editing launch scripts or config files by hand.
📋
Live Launch Log
Watch every step of the launch process in a real-time terminal-style log. Know exactly what's happening.
🌑
Dark Theme UI
A carefully designed dark interface with an animated glow button. Easy on the eyes at any hour.
☕
Cross-Platform
A single .jar file runs anywhere Java 17+ is installed — Windows, macOS, or Linux. No installation wizard needed.

Get Started

Up and running
in 3 steps.

Install Java
CraftLaunch needs Java 17 or newer. Download it free from Adoptium if you don't already have it.
https://adoptium.net/
Download the JAR
Download CraftLaunch.zip from the latest release, then extract it anywhere on your computer.
CraftLaunch.zip (~70 KB)
Run It
Double-click the JAR, or launch from the terminal. Windows users can use the included launch.bat.
java -jar CraftLaunch.jar

Ready to Play?

Download CraftLaunch, pick your version, and jump in.
